# Monarq Brand Kit

This repository contains the brand assets and design system for Monarq. The brand kit includes essential design elements that ensure consistency across all Monarq products and marketing materials.

## Contents

The brand kit includes:

- Monarq logos
- Illustrations
- Wallpapers
- Templates

All brand assets and design elements are proprietary and owned by Monarq. Unauthorized use or distribution is prohibited.
